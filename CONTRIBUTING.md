# Contributing to Winaborator

Thanks for trying Winaborator! Since this is an early beta, the most valuable thing you can do right now is **report bugs**.

---

## Reporting a Bug

1. Go to [Issues](https://github.com/jakeefr/winaborator-ai/issues)
2. Click **New Issue**
3. Fill out the bug report template

### What to include

- **Windows version** (e.g. Windows 11 22H2)
- **Winaborator version** (shown in the title bar or About section)
- **What you were doing** when the bug happened
- **What you expected** to happen
- **What actually happened**
- **Screenshots or screen recordings** if possible
- **Steps to reproduce** if you can reliably trigger it

---

## Feature Requests

Feature requests are welcome — open an issue and label it as a suggestion. No guarantees, but all feedback is read.
